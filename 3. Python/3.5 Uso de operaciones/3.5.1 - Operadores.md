# Â¿Cuales son los operadores en Python?

* Suma = + 
```
resultado = 5+10
print(resultado)
```

* Resta = - 
```
resultado = 5-10
print(resultado)
```

* Multiplicacion = *
```
resultado = 5*10
print(resultado)
```

* Division = / 
```
resultado = 5/10
print(resultado)
```

    *Division a entero = //
    ```
    resultado = 5//10
    print(resultado)
    ```

Las operaciones se evaluan segun un orden:
1. Parentesis
2. Exponentes
3. Multiplicacion y division
4. Suma y resta

## Uso de numeros en python

* Convertir cadenas en numeros
```
resultado = int("123")
print(resultado)
```

* Valor absoluto
```
resultado = abs(20-30)
print(resultado)
```

* Redondeo
```
resultado = round(3.6)
print(resultado)
```

* Biblioteca matematica
```
from math import ceil,floor
resultado1 = ceil(12.5)
print(resultado1)
resultado2 = floor(12.5)
print(resultado2)
```
ceil redondea al entero mayor mas cercado
floor redondea al entero menor mas cercano
